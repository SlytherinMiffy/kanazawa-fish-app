# Web クイズテンプレート

難易度・選択肢数に対応した Web クイズのテンプレートです。

## 使い方

1. `index.html` をブラウザで開いてクイズを開始します
2. スタート前に **難易度**（簡単・普通・難しい）を選択できます
3. 問題を追加・編集する場合は `quiz.js` の `allQuestions` 配列を編集してください

## 問題の追加方法

`quiz.js` の `allQuestions` 配列に以下の形式で追加します：

```javascript
{
  question: '問題文',
  correct: '正解',
  distractors: ['不正解1', '不正解2', '不正解3', ...],  // 候補（ここからランダムに選択）
  choiceCount: 4,  // 表示する選択肢の数（3・4・5）
  difficulty: 'easy',  // 'easy' | 'normal' | 'hard'
  explanation: '正解の解説文'
}
```

- **correct**: 正解
- **distractors**: 不正解の候補（表示時にランダムに選択され、毎回違う選択肢が出る）
- **choiceCount**: 選択肢の数
- **explanation**: 回答後に表示する解説

## ファイル構成

- `index.html` - メインの HTML 構造
- `styles.css` - スタイル（ダークテーマ）
- `quiz.js` - クイズのロジックと問題データ

## 機能

- 難易度（簡単・普通・難しい）でフィルタ
- 進捗バー表示
- 正解/不正解と解説を表示
- 前へ / 次へ ナビゲーション
- スコア表示と結果メッセージ
- もう一度挑戦
- レスポンシブデザイン
