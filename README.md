# 金沢おさかなアプリ (Frontend)

このプロジェクトは、金沢の美味しい魚とお店を紹介し、ユーザーが楽しく巡れるアプリのプロトタイプです。

## 機能
- **ユーザー認証**: 登録・ログイン (現在はMockモードで動作)
- **ダッシュボード**: 訪問状況やポイントの確認
- **店舗リスト**: お店の一覧とプレースホルダー地図
- **おさかなクイズ**: クイズに答えてポイントゲット

## 始め方

### 1. 依存関係のインストール
ネットワーク環境が安定している状態で以下を実行してください。
```bash
npm install
```
※もしエラーが出る場合は、プロキシ設定やネットワーク接続を確認してください。

### 2. アプリの起動
```bash
npm run dev
```
ブラウザで `http://localhost:5173` を開くとアプリが利用できます。

## 注意事項 - Mockモード
現在、PHPバックエンドが利用できない環境向けに、認証機能はブラウザの `localStorage` を使用した **Mockモード** で動作しています。
- 登録したユーザー情報はブラウザ内に保存されます。
- サーバー（PHP）は不要で動作します。

## 構成
- **Frontend**: React, TypeScript, Tailwind CSS
- **Routing**: Internal State Router
